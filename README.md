# Portfolio React Starter

Simple React + TypeScript portfolio starter built with Vite.

## Commands

- `npm install` - install dependencies
- `npm run dev` - start local development server
- `npm run build` - build for production
- `npm run preview` - preview production build

## Starter Sections

- Hero
- About
- Projects
- Contact

Edit content in `src/App.tsx` and styles in `src/style.css`.
